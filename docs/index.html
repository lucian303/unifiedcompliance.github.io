Getting Started

The CCH API allows you to programmatically access many of the features and data available in the Common Controls Hub.

The API supports use cases for large enterprise software offerings, internal applications, and quick one-off interfaces for internal initiatives.

Documentation

We have different sets of documentation depending on your needs:

  • Our complete API documentation - If you like to read the manual before using your TV. We have plenty of examples.
  • API FAQs - If you're pressed for time, and want to get up to speed quickly.
  • Try it Out! - If you want to just start tinkering. Scroll down to the "Try it Out!" section to start making API calls right from this page!

Key Points for Getting Started with the API

  • You must have a CCH Account to use the API. The account can be either a starter account or one with a basic subscription.
  • In order for a list to appear via the API, it must be marked as "shared".
  • UCF data is only accessible through Authority Documents Lists. For example, you cannot query for a list of all Common Controls, you must instead query for all Common Controls associated with a User's Authority Document List.
